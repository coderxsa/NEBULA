1. Download the file.  
2. Open Command Prompt and navigate to the `nebula` folder:  
3. Make sure Node.js is installed. If not, download it from [nodejs.org](https://nodejs.org/).  
4. Install dependencies:  
5. Edit `bot.js` and add your phone number.  
6. Start the bot:  node index.js
